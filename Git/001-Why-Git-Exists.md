# Git 001 - Why Git Exists

## 1. What problem does Git solve?
Git keeps track of the evolution of a software project, whether it is developed by one person or by a team.

This makes it possible to recover previous versions, understand when and why changes were made, and identify where errors were introduced.

## 2. What is a repository?
A repository is a project managed by Git. It contains the project's files and the complete history of every committed change.

## 3. What does it mean that a file is *untracked*?
An untracked file is a file that exists in the project but is not yet being tracked by Git.

## 4. What does `git add` do?
The `git add` command stages changes so they will be included in the next commit.

## Summary
Git allows developers to safely track the evolution of a project through a history of commits.
